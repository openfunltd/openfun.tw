ABOUT US

歐噴這名字，取自於 Open 這個單字，公司創辦人 Ronny 過去參與許多開放政府（Open Goverement）運動，長期推動開放資料（Open Data），並且使用及貢獻開放原始碼（Open Source）。創立歐噴公司，希望能將開放文化變成一家公司，召集有志一同的夥伴取之於開放，用之於開放，一起能在台灣推動開放文化。

英文名稱 OpenFun ，取自於「開放」兩字拆解後會變成「開」+「放」，就是 Open + Fun。我們相信實踐開放文化的過程是有趣的，開放原始碼讓世界上每個人都可以把心中的想法和對新技術的實作轉化為程式碼供別人使用，讓我們可以隨時都可以觸碰到最新技術，並將技術變成應用，我們也可以將我們的成果分享給別人，這樣的過程能不有趣嗎？

創辦人 Ronny 參與 g0v 零時政府十幾年，並發起了「台灣公司資料」、「公司關係圖」、「開放政府標案」、「開放政治獻金」、「求職小幫手」、「新聞小幫手」、「鄉民看電視」、「國會觀察家」...等專案，十幾年來探索資料爬蟲、開放資料、地理資料處理、搜尋引擎、資料視覺化、群眾外包、瀏覽器套件、影音處理、語音轉逐字稿、大型語言模型文字摘要等技術，這些專案其中許多也不只是黑客松期間的展示型專案，而是已經運作超過十年並且到現在仍在持續更新資料並提供服務，成立歐噴公司也是希望能將這些技術，為客戶帶來可以長久穩定運作的系統。

WORKS

台灣公司資料
COMPANY g0v

2013年1月開始寫爬蟲，耗費三個月時間抓取完 200 萬家公司資料，之後建立起每日自動更新機制，持續更新至今，為台灣最完整公開公司資料，並且有使用 API 供人使用，衍生出像台灣公司關係圖、手開發票小幫手等功能，也有 Simpany 及許多會計事務所串接使用。資料庫採用 MySQL + ElasticSearch。

link https://company-graph.g0v.ronny.tw
開放立法院
Legislature g0v

立法院每年產生一萬多則議案，每週召開十數場會議，每月公報產生數萬字文字的會議紀錄，原始格式從 Word、網頁、PDF 不等並分散在各種來源，本公司利用多隻爬蟲進行自動化資料清理匯入，並製成 API 以及相關服務，幫助公民監督國會聯盟自動化立委評鑑及法案對照表瀏覽，以及幫助報導者產製「你家立委4年來為哪些議題發聲？」報導。

link https://openfunltd.github.io/law-diff/
